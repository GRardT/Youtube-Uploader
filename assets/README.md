# Assets Folder

This folder contains application icons and images.

## Required Icon File

**File**: `icon.png`

**Requirements**:
- **Format**: PNG (with transparency support)
- **Size**: 256x256 pixels (will be automatically scaled for different uses)
- **Background**: Transparent or solid color
- **Content**: Your custom icon image

## Usage

The icon is used for:
1. **Window icon** - Shows in taskbar and title bar
2. **System tray icon** - Shows in notification area
3. **Future .exe icon** - When packaged as executable

## How to Add Your Icon

1. Prepare your icon as a 256x256 PNG file
2. Name it exactly: `icon.png`
3. Place it in this `assets/` folder
4. Restart the application

If `icon.png` is not found, the app will use default icons.
