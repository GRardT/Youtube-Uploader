# Version Notes

# google-api-python-client 2.108.0:
#   - Stable version with YouTube Data API v3 support

#

#   - OAuth 2.0 flow for desktop applications
#   - Browser-based authorization
#
# google-auth-httplib2 0.1.1:
#   - HTTP transport adapter for Google API client
#
# pillow 10.1.0:
#   - Image library for creating tray icon
#   - Pure Python, no external dependencies on Windows
#
# pystray 0.19.5:
#   - System tray icon support
#   - Cross-platform, but we use Windows-specific features
#
# requests 2.31.0:
#   - HTTP library for connectivity checks
#   - More reliable than urllib for our use case
#
# pywin32 306:
#   - Windows-specific APIs (file permissions, shutdown handling)
#   - CRITICAL for security features (token file permissions)
#
# win11toast 0.35:
#   - Windows 10/11 toast notifications (actively maintained fork)
#   - Fixes WPARAM errors from unmaintained win10toast
#   - Used for upload success/failure notifications
#
# winshell 0.6:
#   - Windows shell integration
#   - Used for startup folder integration (auto-start with Windows)
#
